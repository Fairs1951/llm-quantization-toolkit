# LLM Quantization Toolkit

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![HuggingFace](https://img.shields.io/badge/HuggingFace-Transformers-orange.svg)](https://huggingface.co/)

Optimize Large Language Models for edge deployment.

## Features
- 4-bit NormalFloat (NF4) support
- 8-bit Integer Quantization
- Automated benchmarking on custom datasets
- Integration with HuggingFace PEFT

## Installation
```bash
pip install -r requirements.txt
```
